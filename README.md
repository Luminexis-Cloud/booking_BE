# AgendaAppointmentBackend

## Quick Start

```bash
npm install
npm run dev
```

## View Database

```bash
npx prisma studio
```
Open: `http://localhost:5555`


## Database Commands

```bash
# Update database schema
npx prisma db push

# Reset database (careful!)
npx prisma db push --force-reset

# View database
npx prisma studio
```

## URLs
- **Backend API:** `http://localhost:3000`
- **Database Viewer:** `http://localhost:5555`
